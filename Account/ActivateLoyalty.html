By creating online user ID, you will be able to do online booking when log in to your account from web

In order to create user ID you have to be user Urgoo Loyalty card. Loyalty card can be purchased from any of “Urgoo cinema” branches.

To create online user ID you need to input following information (Please note that your e-mail address should match the registered e-mail address in the cinema database):

-You loyalty card number (Shown on the card)
-Your e-mail address. If you do not remember you email please insert the pin code
-Please insert new user ID and password when you login to your account from this website..

if you not insert e-mail address, or your e-mail does not match with the initially registered e-mail address, those cases you are not able to create user ID. If your e-mail address is not matching please contact to our call center 77117711.

If you cannot access to your address please contact or send us e-mail to info@urgoo.mn.
